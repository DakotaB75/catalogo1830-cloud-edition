# Code of Conduct

## Our Commitment
This project is maintained as a professional cloud engineering portfolio. We are committed to a respectful, constructive, and technical collaboration environment.

## Expected Behavior
- Constructive feedback
- Professional communication
- Respect for contributors and maintainers
- Focus on engineering quality and improvement

## Unacceptable Behavior
- Harassment or disrespectful communication
- Disruptive or malicious activity
- Misuse of issues or pull requests

## Enforcement
Maintainers reserve the right to remove, reject, or restrict contributions that do not align with this Code of Conduct.
