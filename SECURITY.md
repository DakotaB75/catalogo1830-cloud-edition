# Security Policy

## Supported Versions
This repository is maintained as a cloud engineering portfolio and demo environment. Only the latest version is actively supported.

## Reporting a Vulnerability
If you identify a security issue, please do not open a public issue.

Instead, report it privately via GitHub Security Advisories or contact.

All reports will be reviewed and addressed appropriately.

## Scope
This repository includes:
- Azure deployment configuration
- CI/CD pipelines (GitHub Actions)
- Containerized workloads (Docker)
- Infrastructure documentation

Unauthorized exploitation or misuse of any deployed resources is strictly discouraged.
